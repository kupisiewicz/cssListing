Toolbox with text selecting and style switching

Buttons' container can be visible at all time or slide from the top while hover event is called. Any other ways of showing this element can be achieved by setting proper CSS. 

.jQueryCSSListing .buttons { position: absolute; top: 0px; right: 0px; text-align: right; -webkit-transition: top 0.33s 0.1s ease-out; -moz-transition: top 0.33s 0.1s ease-out; -o-transition: top 0.33s 0.1s ease-out; transition: top 0.33s 0.1s ease-out; top: -30px; }
.jQueryCSSListing.fadeButtons .buttons { position: absolute; top: 0px; right: 0px; text-align: right; -webkit-transition: opacity 0.53s 0.1s ease-out; -moz-transition: opacity 0.53s 0.1s ease-out; -o-transition: opacity 0.53s 0.1s ease-out; transition: opacity 0.53s 0.1s ease-out; opacity: 0; -moz-opacity: 0; -webkit-opacity: 0; }
.jQueryCSSListing.fadeButtons:hover .buttons { opacity: 1; -moz-opacity: 1; -webkit-opacity: 1; }
.jQueryCSSListing.slideButtonsRight .buttons { position: absolute; top: 0px; right: -100%; text-align: right; -webkit-transition: right 0.33s 0.1s ease-out; -moz-transition: right 0.33s 0.1s ease-out; -o-transition: right 0.33s 0.1s ease-out; transition: right 0.33s 0.1s ease-out; }
.jQueryCSSListing.slideButtonsRight:hover .buttons { right: 0px; }
